# Iris-Flower-Classification # Iris Flower Classification

This project uses the Iris dataset from Scikit-learn to classify iris flowers using a Random Forest Classifier.

## Features
- Loads the Iris dataset
- Displays dataset information
- Splits data into training and testing sets
- Trains a Random Forest model
- Predicts flower species
- Displays accuracy, classification report, and confusion matrix
- Visualizes the confusion matrix using Seaborn

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Installation

```bash
pip install pandas matplotlib seaborn scikit-learn
```

## Run

```bash
python iris_classification.py
```

## Dataset

The dataset is loaded directly from Scikit-learn:

```python
from sklearn.datasets import load_iris
```

## Output

- Model Accuracy
- Classification Report
- Confusion Matrix
- Prediction for a new sample
